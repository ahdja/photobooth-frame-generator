Photobooth Frame Generator Example

A simple demonstration of how the Photobooth Frame Generator detects transparent slots and fills them with photos.

Detect Slots Only

Select a frame image to detect transparent slot coordinates without requiring user photos.



1. Select Frame (PNG with transparent slots)

2. Select Photos (Select multiple files)

The order of selection determines which photo goes into which detected slot (top-left to bottom-right).

3. Render Mode



In assigned mode, the first selected photo is placed into the chosen slot index. Use the detect tool above to inspect slot numbers.
